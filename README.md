# flutter_chatapp
QTCHAT APP (FLUTTER) (Firebase, Riverpod 2.0) 

## Features
+ Login, Sigin with phone number 
+ Select contact, image on your device 
+ Realtime chat and oder by time sent
+ Online & offline status
